Parameters:

stream – The binary stream to read from.

Returns:

A tuple of (version, checksum).

inline void read_body(BinaryStream &stream, bool is_memberwise)#

Read the body of the map from the stream. First reads the size (uint32_t) of the map, then calls m_key_reader and m_value_reader to read the keys and values. If member-wise, reads all keys first, then all values. Otherwise, reads key-value pairs one by one.

Parameters:

  • stream – The binary stream to read from.

  • is_memberwise – Whether the current reading mode is member-wise.

inline virtual void read(BinaryStream &stream) override#

Read a map from the stream. Reads a fNBytes+fVersion header, then reads element version/checksum via read_element_version(), and finally calls read_body() to read the map body.

Parameters:

stream – The binary stream to read from.

inline virtual uint32_t read_many(BinaryStream &stream, const int64_t count) override#

Read multiple maps from the stream. If m_with_header is true, reads a fNBytes+fVersion header and element version/checksum once before reading multiple maps.

Parameters:

  • stream – The binary stream to read from.

  • count – Number of maps to read. If negative, reads according to the fNBytes header.

Returns:

Number of maps read.

inline virtual uint32_t read_until(BinaryStream &stream, const uint8_t *end_pos) override#

Parameters:

  • stream – The binary stream to read from.

  • end_pos – The end position to stop reading.

Returns:

Number of maps read.

Number of maps read.

class TObjectReader : public uproot_custom::IReader#

Reader for TObject.

Public Functions

inline TObjectReader(string name, bool keep_data)#

Construct a new TObjectReader object.

Parameters:

  • name – Name of the reader

  • keep_data – Whether to keep the read data

inline virtual void read(BinaryStream &stream) override#

Read a TObject from the stream. A TObject contains fVersion (int16_t), fUniqueID (int32_t), fBits (uint32_t). If fBits & kIsReferenced, then a pidf (uint16_t) follows. If m_keep_data is true, the read data will be stored.

Parameters:

stream – The binary stream to read from

inline virtual py::object data() const override#

Get the data read by the reader. This should be called after the whole reading process.

Returns:

If m_keep_data is true, returns a tuple of numpy arrays: (unique_id, bits, pidf, pidf_offsets). Otherwise, returns None.

Private Members

const bool m_keep_data#

Whether to keep the read data.

SharedVector<int32_t> m_unique_id#

Store fUniqueID values.

SharedVector<uint32_t> m_bits#

Store fBits values.

SharedVector<uint16_t> m_pidf#

Store pidf values.

SharedVector<int64_t> m_pidf_offsets#

Store offsets for pidf.
